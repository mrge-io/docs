# mg help

## Name

`mg help` â€” Print usage information for any mrge command.

## Synopsis

```bash
mg help [<command_name>]
```

## Description

- Without arguments, displays a list of common mrge commands.
- With `<command_name>`, shows detailed usage for that specific command.

## Examples

```bash
# Show general help
mg help

# Show help for mg split
mg help split
```

## Notes / Caveats

- Also supports `mg --help`.
- For man-page style references, browse the [Command Reference](/reference/mg-init).
