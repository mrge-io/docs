## Name

`mg help` â€” Print usage information for any cubic command.

## Synopsis

```bash
mg help [<command_name>]
```

## Description

- Without arguments, displays a list of common cubic commands.
- With `<command_name>`, shows detailed usage for that specific command.

## Examples

```bash
# Show general help
mg help

# Show help for mg split
mg help split
```

  - Also supports `mg --help`. - For man-page style references, browse the
